static Daemon\BaseDaemon::toCommand (   $appPath)
static

Returns a shell command that can start this daemon.

This will return the correct result when called on subclasses of BaseDaemon; it is not necessary to reimplement it. It has no purpose within BaseDaemon itself, it's only useful for daemon classes that extend this class.
