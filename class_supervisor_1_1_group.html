Detailed Description

Dynamically controlled Supervisor process group.

Access a Group via its service name, e.g. fiendish.groups.mygroupname.

Supervisor\Group::newProcess (   $namePrefix,
  $command,
  $arg = null 
)

Creates a new Process.

New processes will not actually be started until you call flush().

Processes will be passed an additional argument on the command line, a JSON-encoded data structure with the following keys:

  • groupName - The name of the containing Supervisor group.
  • procName - The name of the process within Supervisor, not including the group prefix.
  • arg - The $arg argument passed to newProcess.
Parameters
$namePrefixA short string describing this Process; can be non-unique. The generated process's name will also include a unique random suffix.
$commandThe shell command that the Process will run. If you are running a daemon implemented in PHP as a subclass of Daemon\BaseDaemon then you can get an appropriate command by calling the static method getCommand() on your class.
$arg(Optional) JSON-encodable object to pass to process as an argument. Note that associative arrays are converted into objects due to passing through PHP's somewhat odd JSON decoding conventions.
Returns
The new Process object. It's also tracked internally by the Group, so you don't need to worry about saving this return value. However, you will want need to preserve the name if you want to get this specific Process out of the Group later with getProcess().
